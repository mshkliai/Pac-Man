# Pac-Man
Arcade Game
***
<https://en.wikipedia.org/wiki/Pac-Man>

***

#### Written on C++11 with graphic library SFML

***
### ***Usage:***

* `git clone https://github.com/mshkliai/Pac-Man.git`
* `cd Pac-Man`
* `make`
* `./pacman maps/map`
